# SnowTab

SnowTab is a jolly focus homepage with quick links, a simple todo list, a Pomodoro timer, a Christmas countdown, snow, lights, and illustrated Christmas wallpapers.

## Features

- Search bar for URLs or web searches
- Editable quick links
- Simple todo list
- Focus task support
- Simple Pomodoro timer
- Compact Christmas countdown
- Settings for timer seconds, clock seconds, themes, snow, lights, and wallpapers
- 24 illustrated Christmas wallpapers
- Favicon included
- No framework and no build dependencies

## Run locally

```bash
pnpm install
pnpm dev
```

Then open:

```txt
http://localhost:5173
```

## Build

```bash
pnpm build
```

The static files will be copied to `dist`.

## Deploy

You can deploy this as a static site on GitHub Pages, Netlify, Vercel, or any static hosting service.
