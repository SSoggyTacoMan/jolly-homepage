# SnowTab

A clean Christmas focus homepage with a big clock, search, quick links, a simple todo list, a simple Pomodoro timer, a Christmas countdown, and wallpaper options.

## Run

```bash
pnpm install
pnpm dev
```

Open:

```txt
http://localhost:5173
```

## Build

```bash
pnpm build
```

The static build is copied to `dist/`.

## Features

- Big clock and date
- Large search box
- Quick links with favicons
- Simple todo list
- Simple Pomodoro timer
- Compact Christmas countdown
- Settings for themes, wallpapers, snow, lights, and timer length
- Built-in SVG Christmas wallpapers
- No build dependencies
